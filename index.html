Varun's Indian Master Plan Viewer

      About

      Master Plan Viewer is a lightweight web application that displays georeferenced raster master plans of Indian cities on an interactive map. It uses Cloud Optimized GeoTIFFs (COGs) for efficient streaming and OpenLayers for mapping.
